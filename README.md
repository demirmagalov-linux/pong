# Pong

A single player Pong game built with Python and Ursina Engine. Play against a CPU opponent and first to 5 points wins.

## How to play

Run the game with:

python pong.py

Controls:

- W: move paddle up
- S: move paddle down

First player to score 5 points wins.

## Features

- CPU opponent that tracks the ball with imperfect movement
- Ball speeds up after each paddle hit
- Ball resets in a random direction after each point
- Win condition with a game over screen

## Requirements

Python 3 and Ursina Engine.

To install Ursina:

pip install ursina

## Notes

The game is single player. The right paddle is controlled by the CPU. Difficulty increases naturally as the ball speeds up with each rally.
